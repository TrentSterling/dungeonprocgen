# Dungeon ProcGen Tutorial

A comprehensive guide and visualizer for various procedural generation methods used in dungeon creation.

## Project Structure
- `index.html`: The hub page.
- `styles/`: Global styles.
- `scripts/`: Global scripts and shared utilities.
- `methods/`: Individual tutorials and visualizers for each method.
  - `random-walk/`
  - `cellular-automata/`
  - `bsp/` (Binary Space Partitioning)

## Tech Stack
- HTML5 / CSS3
- Vanilla JavaScript
- GitHub Pages for hosting
