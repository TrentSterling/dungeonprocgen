Voronoi Diagrams (Biomes)

The Concept

A Voronoi Diagram partitions a plane into regions based on distance to points (sites). For any pixel, its region is determined by the closest site.

Game Dev Use: Generating biomes (forest, desert, snow), political maps, or organic cell structures.

Click canvas to add points!

Core Logic (Pixel Shader Style)


// For every pixel (x,y)
let closestDist = Infinity;
let closestSite = null;

sites.forEach(site => {
    let d = dist(x, y, site.x, site.y);
    if (d < closestDist) {
        closestDist = d;
        closestSite = site;
    }
});

pixelColor = closestSite.color;
